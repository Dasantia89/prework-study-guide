Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element contains information about the author, copyright information or a sitemap
  • The paragraph element makes formatted text into a paragraph
  • The line break element creates a line break on the web page

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class is used to change the styling of only certain elements to which the class is assigned
  • Styling can be used inline, in the head, or in an external file
  • If using an external file for css it must be linked in the head

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main downloads the files from the main branch and updates local files
  • git init initiates the current directory as a git repository
